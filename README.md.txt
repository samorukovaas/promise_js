

# Можете ли вы повторить Async.js?

При выполнении асинхронного программирования вы часто захотите выполнить несколько
операции параллельно. В некоторых случаях вы можете отложить дальнейшую обработку
пока список асинхронных операций не будет завершен.

В синхронном коде это тривиально, потому что наши операции выполняются в
В порядке их указания:

```js
const thingOne = getThing(1);
const thingTwo = getThing(2);

combine(thingOne, thingTwo);
```

Необходимо реализовать функцию таким образом, чтобы мы могли указать список
асинхронных значений, которые мы хотели бы получить, а затем использовать один раз 
сразу как они станут доступны

```js
getAll(fetch(1), fetch(2))
  .then(function (values) {
    console.log(values[0], values[1]);
  });
```
